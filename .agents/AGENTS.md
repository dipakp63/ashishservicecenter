# Standing Instructions

1. **Responsibility Scope**: You are responsible for the implementation of ideas/modules on the code level only.
2. **Verification Scope**: Verification of implementation or changes must be done on the code level only (e.g. running unit tests, integration tests, or API queries).
3. **UI Verification**: UI-level verification will be done by the End User only. Do not attempt browser-based/UI-based visual tests unless explicitly asked by the user.
